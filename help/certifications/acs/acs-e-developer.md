# 认证历程Adobe [!DNL Campaign Standard] 开发人员专家

了解如何成为经认证的Adobe Developer专家。

## 最低体验

您应该：

* 至少两年 [!DNL Campaign] 软件体验
* 在多渠道数字营销活动和Web开发（例如，REST API）中至少拥有两年的相关经验
* 能够在Adobe Campaign中执行任务，包括模板开发、技术工作流、模式开发和集成，并且能够使用控制面板监控实例
* 在渠道投放环境（电子邮件、直邮、移动设备和社交）中工作的体验
* 熟悉多接触、生命周期和重新激活等营销活动策略
* 了解营销量度，如投资回报率、点进率和转化率。
* 管理营销自动化平台的经验
* CSS、HTML和JavaScript、电子邮件的基本知识
* 关系数据库与数据建模的基础知识
* 对技术工作流的了解和配置
* 扩展收件人表或模式和数据管理的体验
* 创建分类规则、工作流、投放和营销活动模板
* 配置的角色和权限
* 使用管理控制台以及与其他Adobe应用程序集成的体验
* 对Adobe I/O和Adobe I/O Runtime的认识
* 在多渠道上处理事务型消息传递和消息投放的体验
* 使用受管资源包的体验

>[!NOTE]
>
>您的证书有效期为2年，从您通过之日算起。 我们建议尽可能使用较新版本，因为它涵盖较新的产品功能。

**考试目标和范围**

第1节：数据建模和管理(26%)

* 在某种情况下，将客户端需求转换为自定义资源
* 确定如何配置屏幕和过滤器定义
* 在给定情景下，确定定义键值、索引和链接的正确策略
* 确定发布资源更新的正确方法
* 数据建模问题疑难解答
* 确定构建技术工作流的正确方法

第二节：营销活动管理(50%)

* 工作流错误疑难解答
* 确定构建营销活动的正确方法
* 确定正确的受众和用户档案处理方法
* 通过分类应用和创建过滤器规则
* 确定创建和管理模板的正确方法
* 确定配置登陆页面的正确方法
* 确定正确的服务和订阅使用方法
* 在此情景下，验证投放机制
* 确定使用Creative Designer构建电子邮件的正确方法
* 确定创建自定义报告的正确方法

第三节：管理(24%)

* 确定创建用户安全组和组织单位的正确方法
* 确定管理和部署资源包的正确方法
* 确定配置外部帐户的正确方法
* 确定配置选项的正确方法
* 确定创建品牌的正确方法
* 确定开发渠道配置的正确方法

**参加练习测试**

我们建议您在开始之前参加免费练习测试。 您将可以不受限制地访问Adobe专家级考试的所有可用练习测试。

我们的实践测试都是从与实时测试相同的蓝图中开发的。 练习测试将帮助您了解实时考试中涵盖的主题。

请记住，通过练习测试并不保证您会通过认证考试。

转到 [Adobe认证考试准备门户](https://www.certmetrics.com/adobe/candidate/gmetrix_sso.aspx){target="_blank"} 现在去做练习测试。

**参加认证考试**

Adobe使用两家供应商来管理受监控的考试，即PSI和Examity。

* PSI提供测试中心和日语考试选项（如有）
* 示例提供远程联机选项

要计划、重新安排或取消您的考试，只需单击所选供应商，然后按照屏幕上提供的说明操作即可。 很简单！
