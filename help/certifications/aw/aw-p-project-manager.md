# 认证历程Adobe [!DNL Workfront] 项目经理专业人员

了解如何成为经认证的Adobe项目经理专业人员。

## 最低体验

您应该：

* 从最初的工作请求到完成，或接触到项目或工作管理，对工作流程的了解
* 完成了自我指导培训或训练集，或是 [!DNL Workfront] 用户6-12个月，以便他们了解基本功能
* 了解如何配置、跟踪和监视报表数据

>[!NOTE]
>
>您的证书有效期为2年，从您通过之日算起。 我们建议尽可能使用较新版本，因为它涵盖较新的产品功能。

>[!BEGINTABS]

>[!TAB 准备就绪]

**目标受众：**

* 项目经理，计划员工作经理
* SCRUM主控
* [!DNL Workfront] 电源用户
* 系统管理员
* 项目经理，项目组合经理
* Director，经理
* 资源管理器，流量管理器

**考试详细信息 — Adobe [!DNL Workfront] 项目经理**

* 级别：专业（0-12个月经验）
* 通过分数：45/70
* 时间：140分钟
* 投放：在线监控（需要摄像头访问）
* 可用语言：英语
* 成本：125美元（全球）/ 95美元（印度）
* 考试ID:AD0-E903

**准备情况自评**

完成 [在线准备调查表](https://scorpion.caveon.com/launchpad/ad-q-e903-readiness-questionnaire-for-adobe-workfront-project-manager-professional/ad-q-e903-readiness-questionnaire-for-adobe-workfront-project-manager-professional){target="_blank"} 以查看您当前的体验是否与建议的最低候选资格匹配。

**考试目标和范围**

第1节：入学和规划(30%)

* 确定接收工作请求通知的不同方式
* 给定请求后，请查看自定义数据以确定相应的操作
* 在某种情况下，确定活动所需的共享权限
* 在给定方案下，确定适当的任务设置以计划在特定时间范围内执行任务
* 确定在项目中查找关键数据的方法
* 确定跟踪项目风险的不同方式
* 在中确定对象的正确层次结构 [!DNL Workfront]
* 确定用于查看特定分配资源的相应字段
* 给定角色，确定在项目/工作流程的工作进入或规划阶段对角色有用的摄取或规划报表/功能板

第二节：执行(50%)

* 确定了解、应用和衡量项目里程碑的方法
* 创建 [!DNL Workfront] 单次使用批准和何时使用批准
* 识别系统如何使用不同的日期类型
* 在某种情况下，确定日期类型向项目经理传递的信息
* 在给定方案下，确定使用时不同的任务约束对项目时间轴有何影响
* 在给定的情景中，演示如何查找项目团队以及如何通知他们工作分配、更新或完成工作项
* 在给定的情景中，描述如何上传文档并生成校样
* 确定管理项目的不同方式
* 在给定方案下，确定单个项目中特定任务职责的计划差异
* 在此情景下，在 [!DNL Workfront]
* 给定甘特图，解释图表上的对象关系
* 比较条件和进度状态之间的差异
* 识别方式 [!DNL Workfront] 团队和 [!DNL Workfront] 可使用群组
* 在此情景中，描述如何向利益相关方提供所需的信息
* 在给定的方案中，确定日期类型和持续时间对项目条件的影响情况
* 如果出现这种情况，请确定需要自定义状态
* 确定 [!DNL Workfront]
* 在某种情况下，描述构建自定义报表的步骤

第三节：结帐(20%)

* 给定特定对象状态，确定需要执行哪些活动或触发哪些操作
* 在某种情况下，确定报表类型和用于修改任务列表或报表的报表元素
* 识别可在报表中使用的不同通配符
* 确定可用于衡量项目成功与否的项目元素
