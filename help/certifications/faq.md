# Adobe认证常见问题解答

要了解有关Adobe认证计划的更多信息，请参阅下面的常见问题解答(FAQ)。

![横幅](/help/certifications/assets/home_banner_narrow.png)

>[!BEGINTABS]

>[!TAB General]

+++认证有什么好处？

个人优势：认证是一种凭据，用于识别个人在Adobe应用程序中的技能和能力。 通过认证考试的候选人成为Adobe认证专业、专家或主控。

认证需要承诺，在法律上是站得住脚的，并且需要Adobe您的专业知识来验证。 您还可以从下载标记以在社交用户档案中展示您的证书 [!DNL Credly].

详细了解如何在 [!DNL Credly]，转到 [此处](https://experienceleague.adobe.com/docs/certification/certification/how-to-get-certified.html?lang=en){target="_blank"}.

+++

+++我如何知道要参加哪项考试？

认证是基于角色的，这意味着有多项可用的考试。 查看认证历程页面以选择要参加的考试。 通过以下方式选择考试：

* 应用程序：例如， [!DNL Adobe Analytics] 或 [!DNL Marketo Engage]
* 角色：例如，开发人员或业务从业人员
* 级别：例如，专业人士，0-12个月的经验；或专家1-3年

有时，考试会有多个版本。 最好采用最新版本。

* 如果考试即将退休，您将在认证历程页面上看到注释。 这意味着由于有新版本的考试可用，因此该考试版本很快将被删除。 我们鼓励候选者采用最新版本，但如果您更愿意接受，您可以采用即将退休的版本。 无论哪种情况，您的认证将持续两年。
* 如果您当前的证书即将过期，您应续订该证书。 续订选项有较少的问题，并且是免费的。 这些选项涵盖最新的应用程序功能。 如果您的认证过期，您必须参加最新版本的认证考试，您需要为此付费。 [了解有关续订选项的更多信息](https://experienceleague.adobe.com/docs/certification/certification/renew.html?lang=en){target="_blank"}.

+++

+++如果在我的指导考试期间遇到技术问题，该怎么办？

PSI和 [!DNL Examity] 通过聊天、电子邮件和电话获得支持。 Adobe支持无法帮助解决考试期间出现的技术问题。

[!DNL Examity] 支持 - `adobepartnersupport@examity.com`

PSI支持 —  `psisupport@psionline.com`

最快的帮助资源是通过聊天。 所有考试考生在考试前应确保其计算机具有适当的系统要求。

有关更多详细信息，请参阅：PSI和 [!DNL Examity] 用于管理考试和技术设置的页面。

+++

+++我可以与谁联系，以解决认证问题或问题？

有关考试注册、考试付款、测试投放、测试中心、考试评论、考试问题、测试体验、测试适应或有关认证门户的问题，请联系 `certif@adobe.com`.

+++

+++Adobe能否接纳有特殊要求的候选人？

Adobe可以满足某些特殊要求。 如果您需要额外时间完成考试或其他住宿，请联系 `certif@adobe.com` 请你帮忙。

**您的请求必须在您的考试时间表前2周发送。**

+++

+++如果失败，我何时可以再次尝试认证考试？

要获得或保持您的Adobe认证状态，您需要通过最新的认证考试，重新参加。 建议您在认证过期之前留出足够的时间考虑潜在的重新使用。

对于首次尝试Adobe认证考试不及格的应聘者，他们必须至少等待24小时后再次尝试。 如果在第二次或随后的任何尝试中失败，则需要等待14个日历日才能重考。 每次尝试都会产生单独的考试费用。

+++

+++如何获取我的得分报表副本？

* 步骤1:登录到 [Adobe凭据管理系统](https://www.certmetrics.com/adobe/){target="_blank"}、 CertMetrics. 使用您的ID或电子邮件地址和密码登录。
* 步骤2:单击“成绩单”。
* 步骤3:滚动浏览记录历史记录并选择要查看的报表。 您可以通过电子邮件将报表发送给您或在屏幕上查看。

+++

+++如何跟踪我的证书状态？

您可以通过转到 [Adobe凭据管理系统](https://www.certmetrics.com/adobe/){target="_blank"} 在“认证”选项卡下。

+++

+++通过认证考试后能得到什么？

完成考试后，结果会立即显示在屏幕上。 在72小时内，您将从CertMetrics收到对得分的电子邮件确认，并PDF您的证书。 您可以从CertMetrics帐户访问显示您的认证状态的徽标，并可在您的网站、电子邮件、签名、名片、社交资料等上使用。

+++

+++如何登录或创建Adobe凭据管理/Certmetrics帐户？

您可以 [在此处登录或创建帐户](https://www.certmetrics.com/adobe/){target="_blank"}. 您可以在此处查看可用的考试、考试历史记录、认证过期日期，以及任何优势，如折扣。

+++

+++认证持续多长时间？

由于我们需要确保我们的Adobe认证人员对新应用程序特性和功能具有最新的需求，因此Adobe认证是 **有效期为两年**.

+++

+++是否提供英语以外的语言的考试？

通过PSI（我们的供应商之一），我们提供一些日文考试。 如果考试的日文版是可用的，将在该考试的考试详细信息中注明。 测试详细信息位于认证历程页面的“准备就绪”选项卡中。

+++

+++您是否在Adobe活动中提供认证？

在某些Adobe赞助的活动中提供认证。

+++

+++认证是否与特定版本的应用程序(例如，Adobe Experience Manager 6.5)绑定？

不是直接的。 但是，由于认证现在每2年需要续订一次，因此您的认证可确保您是Adobe应用程序最新版本的最新版本。

+++

>[!TAB 通过按需考试续订]

+++什么是认证续订？ 如何续订证书？

Adobe认证在两年后过期。 您可以通过参加并通过最新版本的适用Adobe认证考试来续订您的认证。

您有两个选项：

* 定期参加认证考试
* 参加续订考试 — 一个较短的考试，重点介绍最新的解决方案功能以及您已经了解的内容。

如果您没有原始证书或您的证书已过期，则无法参加续订考试。

我们偶尔会为某个角色退休考试。 在这种情况下，您的认证将持续两年，然后该认证将终止。 例如，我们不再提供Lead/Dev Ops认证，因此认证将在不进行替换测试的情况下过期。

+++

+++如果我在证书过期前未续订证书，会发生什么情况？

您的证书将过期，并在 [!DNL Credly] 和您的任何社交资料。 此外，如果您的认证过期，您将无法参加较短的续订考试或通过持续学习续订。

详细了解如何在 [!DNL Credly]，转到 [此处](https://experienceleague.adobe.com/docs/certification/certification/how-to-get-certified.html?lang=en){target="_blank"}.

+++

+++续订我的证书需要多少费用？

您可以免费续订认证。 这为您提供了一种经济高效的方式，来维护您的认证，并保持您的解决方案技能处于最新状态。

要进一步了解如何续订您的证书，请转到 [此处](https://experienceleague.adobe.com/docs/certification/certification/renew.html?lang=en){target="_blank"}.

+++

+++在哪里可以看到我的证书过期日期？ 我会收到通知吗？

登录到 [Adobe凭据管理系统](https://www.certmetrics.com/adobe/){target="_blank"}. 您可以在此处查看您获得凭据的时间以及凭据过期的时间。 您还应该检查您的用户档案，并确认您的联系人、辅助电子邮件和作业信息正确无误。

您将在认证过期前一个月收到电子邮件通知。 但是，如果您更改了作业，并且未更新电子邮件，则可能不会收到通知。 在您的CertMetrics帐户中添加辅助电子邮件（如Gmail、Hotmail、Yahoo等），以便您获得证书过期提醒电子邮件，即使您更改公司也是如此。

+++

+++我需要参加哪项考试才能续订？

要了解您需要参加哪项考试，请登录 [Adobe凭据管理系统](https://www.certmetrics.com/adobe/){target="_blank"} 使用您的Adobe ID，单击认证，然后选择认证名称。 续订考试列于 **保持当前状态的要求**.

+++

+++续订认证后，我会获得新的数字徽章吗？

不会。您的数字徽章将保持不变。

+++

+++我是否可以参加最初认证时参加的相同考试，以续订我的认证？

不会，如果通过同一考试进行初始认证，则无法再次参加该考试以续期。

+++

>[!TAB 通过持续学习进行续订]

+++通过持续学习，Adobe的认证续订计划是什么？

Adobe提供持续的学习和参与活动，以便候选人能够保持活跃状态并将其认证状态延长2年。 这仅适用于选定的应用程序。

+++

+++我的证书已过期；我是否可以继续开展持续的学习和参与活动，以便续订？

不能，此续订选项仅适用于Adobe经过积极认证的Analytics、Experience Manager、商务和Target候选人。

+++

+++哪些认证可提供持续的学习和参与活动以进行续订？

目前，此计划涉及Analytics、Experience Manager、商务和Target。 我们将通过适用的渠道分享公告，并在我们扩展公告时分享其他认证。

+++

+++哪些持续学习和参与活动是认证续订计划的一部分？

将为候选人提供5至7门按需课程，供其审查。 续订认证的标准是根据课程通过任何3项评估，或通过2项评估，并提交经签署的客户参考表。

+++

+++在认证续订的环境中，什么是客户参考表？

候选人可以提交的表格，作为Adobe持续学习和参与活动的一部分，以续订其证书。 该表格必须由客户签署，说明候选人在最近12个月内在与他们续订其认证的产品相关的项目上至少花费了100小时。 通过将此选项作为认证续订要求的一部分，我们希望确认您持续的参与是验证您的技能和知识的一种方式，以便您的认证保持活跃。

+++

+++通过持续的学习和参与活动进行认证续订是否需要支付费用？

不会。所有活动都可通过Adobe凭据管理器免费访问。

+++

+++我是否需要通过评估并提交客户参考表？

否，提交客户参考表是可选的，候选人可以完成其中一项活动来续订其证书。 例如， [!DNL Adobe Analytics] 认证续订要求从给定列表中至少完成3个活动。 该列表包括通过6项评估并提交客户参考表。 候选人可通过2次评估并提交客户参考表，从而满足续订的3项最低要求。

+++

+++我在一个解决方案中拥有三个认证。 我是否需要完成每项认证的要求？

不需要，您只需完成列出的解决方案要求即可。 完成所需活动并将结果提交到Adobe凭据管理器后，将续订您的证书。 例如，如果您在 [!DNL Adobe Analytics] 作为业务从业者、开发人员和架构师，您只需完成3个活动即可满足所有认证的续订要求。

+++

+++我是需要复习课程，还是干脆开始评估？

评估以Experience League课程为基础，审查这些课程将为候选人成功通过这些课程做好准备。 但是，续订的关键要求只是通过评估。

+++
