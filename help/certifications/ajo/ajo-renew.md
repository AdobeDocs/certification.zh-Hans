# 续订Adobe [!DNL Journey Optimizer] 认证

认证考试合格的，认证有效期为两年。

我们非常高兴地宣布，您现在可以通过一种免费的新形式来续订您的认证。 它涉及使用随选Experience League课程和教程 — 只需完成三项活动即可免费续签您的认证。 您必须在认证过期之前续订，否则需要参加完整考试才能再次获得认证。
