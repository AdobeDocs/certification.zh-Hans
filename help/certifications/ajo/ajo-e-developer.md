# 认证历程Adobe [!DNL Journey Optimizer] 开发人员专家

了解如何成为认证Adobe [!DNL Journey Optimizer] 专家。

## 最低体验

您应该至少拥有一年使用各种CDP和营销自动化系统的经验，并能够执行以下任务：

* 配置用户权限
* 创建和管理沙箱
* 品牌配置
* 了解模式设计概念
* 设置数据源和数据摄取
* 创建区段
* 设置体验事件
* 创建和管理优惠
* 创建模拟（测试优惠）
* 创建电子邮件、短信、应用程序内消息和推送通知
* 使用帮助函数进行内容处理和个性化
* 验证、校对和发布消息
* 创建多渠道历程
* 历程报告
* 测试/验证历程

>[!NOTE]
>
>您的证书有效期为2年，从您通过之日算起。 我们建议尽可能使用较新版本，因为它涵盖较新的产品功能。

**考试目标和范围**

第1节：管理和配置(17%)

* 确定角色以及应用于每个角色的权限
* 描述如何分配/重置/删除沙盒
* 演示对渠道管理的了解
* 演示对事件/操作/数据源的了解

第二节：Adobe Experience Platform基金会(33%)

* 演示对体验数据模型(XDM)的了解
* 说明如何设置数据源和数据摄取
* 演示对分段服务的了解
* 说明如何为AJO配置模式
* 确定AJO上实时客户资料的行为

第三节：offer decisioning(11%)

* 演示对创建和管理优惠和组件的了解
* 演示如何使用测试用户档案运行选件模拟

第四节：内容创作(20%)

* 演示如何在应用程序消息和推送通知中创建电子邮件、短信、
* 使用帮助程序函数演示对内容处理和个性化的了解
* 演示对验证、校对和发布消息的了解

第五节：Journey Orchestration(19%)

* 说明创建多渠道历程的过程
* 确定在何处访问报表功能板
* 演示如何测试和验证历程

**参加练习测试**

我们建议您在开始之前参加免费练习测试。 您将可以不受限制地访问Adobe专家级考试的所有可用练习测试。

请记住，通过练习测试并不保证您会通过认证考试。

转到 [Adobe认证考试准备门户](https://www.certmetrics.com/adobe/candidate/gmetrix_sso.aspx){target="_blank"} 现在去做练习测试。

**参加认证考试**

Adobe使用两家供应商来管理受监控的考试，即PSI和Examity。

* **PSI** 提供测试中心和日语考试选项（如果有）
* **[!DNL Examity]** 提供远程联机选项

要计划、重新安排或取消您的考试，只需单击所选供应商，然后按照屏幕上提供的说明操作即可。 很简单！
