# Adobe的认证历程 [!DNL Marketo Engage] 架构师母版

了解如何成为认证Adobe [!DNL Marketo Engage] 架构师母版。

Adobe凭据程序已启动新Adobe [!DNL Marketo Engage] 为对获得认证母版头衔感兴趣的候选人提供架构师认证。 它取代了以前的Marketo认证解决方案设计师(MCSA)认证面试流程。 [了解详情](https://solutionpartners.adobe.com/solution-partners/news/2022/02/a-new-standard-for-adobe-marketo-engage-certified-architects.html){target="_blank"}

## 最低体验

您应该拥有：

* Adobe中的当前认证 [!DNL Marketo Engage] 商业从业者认证专家
* 至少五年经验最好有多次Adobe [!DNL Marketo Engage]
* 了解业务影响并有效地向不同级别的利益相关者（技术和非技术）解释/传达其解决方案
* 将解决方案概念化并转换为 [!DNL Marketo Engage] 架构，了解受集成影响的业务流程
* 熟悉构建实例的原则

此外，您还应熟悉以下工具和技术：

* Adobe [!DNL Marketo Engage] 特性、功能和应用程序
* 数据库系统
* API连接
* CRM平台
* 营销技术平台

>[!NOTE]
>
>您的认证有效期为从您通过认证后的两年。 我们建议尽可能使用较新的版本，因为它涵盖了较新的产品功能。

**考试详细信息：**

* 先决条件：Adobe Certified Expert - [!DNL Marketo Engage] 商业从业者专家
* 级别：硕士（3-5年经验）
* 从业者认证
* 及格分数：30/50
* 时间：150分钟
* 交付：在线监护（需要摄像头访问）或测试中心监护
* 可用语言：英语
* 费用：225美元（全球）/150美元（印度）
* 考试ID：AD0-E556

**准备情况自我评估**

完成 [在线准备情况调查表](https://scorpion.caveon.com/launchpad/ad-q-e556-readiness-questionnaire-for-adobe-marketo-engage-architect-master-exam/ad-q-e556-readiness-questionnaire-for-adobe-marketo-engage-architect-master-exam){target="_blank"} 查看您当前的体验是否与推荐的最低候选资格相匹配。

**考试目标和范围**

第1款：项目领导(16%)

* 通过案例研究，确定合适的利益相关者
* 评估一组要求并建议项目界限、预算、资源和范围
* 在流程经过修改后，确定哪些人需要启用和有效启用转出计划
* 评估审计工作并建议优先事项和时间表

第2节： [!DNL Marketo Engage] 体系结构设计(48%)

* 收集需求/需求，并推荐一个潜在客户管理和生命周期设计
* 审查现有实施并提供调查结果
* 根据业务需求，识别不同的行为以进行评分
* 审查现有实施，建议采取措施以扩展活动执行
* 概述在多个国家/地区进行销售线索传送的操作过程
* 在情景中，评估将新技术纳入实例的影响和要求
* 确定不同类型的集成以及何时使用它们
* 给定一个方案，概述受提议的更改/要求影响的人员生命周期中的各个阶段
* 评估一组要求并推荐营销活动逻辑的方法

第3节：报告和归因(20%)

* 给出了一个模型以供评述，并解释了生命周期建模器中绕线的概念
* 给定在线广告或自然社交或内容，推荐一种报告方法
* 说明如何验证 [!DNL Marketo Engage] 跟踪生命周期变化中的报告
* 确定要在给定报表类型中使用的量度
* 确定如何跟踪多渠道归因以评估ROI
* 识别并描述不同归因模型的利弊
* 给定业务需求，描述渠道设置对报表的影响

第4款：高级业务(16%)

* 给定场景，确定改进管理或治理的适当步骤
* 给定一组要求，建议一个高效的数据模式
* 评估客户的实施并建议改进可投放性的操作
* 评估客户的实施并提出改进管理操作的建议
* 收集有关客户端实施的信息并推荐安全做法
* 解释给定场景的工作区和分区的适用性
