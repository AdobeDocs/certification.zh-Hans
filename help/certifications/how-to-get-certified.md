# 如何获得认证{#how}

了解如何通过Adobe获得认证。 在获得认证之前、期间和之后，查找您需要的所有资源。

>[!BEGINTABS]

>[!TAB 注册]

Adobe使用两个供应商来管理跟踪的考试： [!DNL PSI] 和 [!DNL Examity].

* **[!DNL PSI]:** 提供测试中心和日语考试选项（如果有）
* **[!DNL Examity]:** 提供远程联机选项

**按照以下步骤注册您的认证考试：**

* **步骤1:** 已成功登录到 [Adobe凭据管理系统](http://www.certmetrics.com/adobe){target="_blank"}.
* **步骤2:** 返回此网站，然后单击技术认证。
* **步骤3:** 选择您的Adobe解决方案、认证级别和角色
* **步骤4:** 按照 **获得认证** 中。

>[!TAB 参加考试]

您的 [个人资料](https://www.certmetrics.com/adobe/candidate/demographics.aspx){target="_blank"} Adobe凭据管理系统中的名称必须与您的政府ID完全匹配。 如果信息不相同或任何不匹配，将导致检查者终止考试，并且除支付额外的考试费用外，还要求您重新安排约会。

如果在预定的考试预约前48小时之前启动，则可能取消和重新安排。 候选人必须自行修改此任命。

Adobe支持或考试供应商都无法处理重新计划和取消请求。 48小时以下的取消除考试费外，还需支付重订费。

完成考试后，您会获得临时通过或失败。 填充最终得分可能最多需要72小时。 您将在 [Adobe凭据管理器](http://www.certmetrics.com/adobe){target="_blank"}.

如果您首次尝试时未通过Adobe认证考试，则必须至少等待24小时，才能重新参加考试。 在第二次考试或随后的任何尝试中未能通过考试，需要等待14个日历日才能重新参加考试。 每次尝试都将产生单独的考试费。

Adobe认证凭单中心(Xvoucher)常见问题解答

+++考试凭单何时过期？ 它们有多长时间？

Adobe考试凭单在购买后12个月到期。 凭单无法延长到过期日期。 考试注册必须在考试凭证过期日期之前计划、付费和参加。

+++

+++凭单是否可退回以贷记或退款？

_所有销售都是最终的。_ 不接受退货和退款。 Adobe考试凭单是在考试级别购买的，而不是在特定的考试中购买，因此具有灵活性。

* 示例：当您购买10个Adobe Certified Expert级考试券并分发它们时，您不必提前知道或计划谁将参加哪个专家级考试。
* 管理员可以将凭单分配给用户，并查看该凭单是否被查看、兑换，或者是否计划或进行了考试。

+++

+++接受哪种付款？

（所有主要）信用卡付款。 通过信用卡付款需要接受在线的点击换行协议。 不同的付款选项包括：

* 电汇、采购订单或ACH:在Xvoucher系统中创建可打印并发送给应付帐款部门的采购订单。 接到电汇或支票后，该订单即获批准。
* 信用卡：所有主要信用卡都接受。
* Xwallet:这将从您的Xwallet池中减去订单金额。

+++

+++我可以购买的凭单数量是否最小或最大？

没有最小值或最大值。 客户购买仅受可用资金（如信用卡限额或电汇的银行账户余额）的限制。

+++

+++下订单后，需要多长时间才能访问凭单？

如果通过信用卡或Xwallet付款，则购买后可立即使用凭单。

* 如果您自己购买，则会在屏幕和电子邮件中提供凭单代码。
* 如果您为他人购买，请登录您的Xvoucher帐户，然后从此处将凭单分发给最终用户。
* 如果通过电汇或采购订单付款，则凭单订单将在Xvoucher收到资金后发放。

[在线帮助](https://help.xvoucher.com/){target="_blank"}

+++

+++如何访问我的Xvoucher帐户？

只需登录到您的帐户即可 [此处](https://www.xvoucher.com/Login.aspx){target="_blank"}.

+++

+++如何访问我的Adobe凭单？

在将凭单分配给您后，您将收到一封来自support@xvoucher.com的电子邮件，其中包含凭单代码以及有关如何兑换凭单的说明。 如果未收到电子邮件，请检查您的垃圾邮件文件夹。 如果您仍然无法找到电子邮件，则可以登录Xvoucher帐户。

+++

+++凭单是否可退回以贷记或退款？

所有销售都是最终的。 不接受退货和退款。 Adobe考试凭单是在考试级别购买的，而不是在特定的考试中购买，这样可以非常灵活。

+++

>[!TAB 管理徽章]

通过认证考试后，您将收到Credly提供的Adobe Experience Cloud数字徽章，Credly是数字凭证服务的提供商。

您的徽章是您熟练使用Adobe Experience Cloud产品的有形证据。 通过在社交媒体用户档案或简历中展示您的成就，您可以向潜在雇主和客户展示您的成就，并展示您在这些应用程序中的技能。

**对于现有用户，有两种方法可接受您的Adobe认证徽章：**

* **步骤1:** 登录 [克雷德利](https://www.credly.com/users/sign_in){target="_blank"}. _(您必须使用与您之前用于获取Adobe凭据的相同电子邮件，否则将看不到您的徽章。)_
* **步骤2:** 或者，转到您从Credly收到的电子邮件，以接受徽章。
* **步骤3:** 接受并共享您的徽章。

**对于新用户，请按照以下说明接受您的Adobe认证徽章：**

* **步骤1:** 转到您从Credly收到的电子邮件以接受该徽章。
* **步骤2:** 创建帐户。 _(您必须使用与您之前用于获取Adobe凭据的相同电子邮件，否则将看不到您的徽章。)_
* **步骤3:** 接受并共享您的徽章。
