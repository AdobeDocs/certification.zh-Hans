# 认证历程Adobe [!DNL Experience Manager Forms] 开发人员专家

了解如何成为认证人员 [!DNL Experience Manager Forms] 开发人员专家。

## 最低体验

您应该：

* 1-3年AEM Forms经验
* 在IT行业至少5年的工作经验
* 能够领导自定义表单解决方案的开发、配置、测试和部署
* 能够配置表单、将业务要求转换为工作流、与外部系统集成
* 了解AEM Forms基础组件

此外，您还应熟悉以下技术和环境：

* 设计XDP模板、XFA模板
* 创建PDF forms（静态/动态、交互式和非交互式）
* 架构或XML数据绑定
* 创建表单片段
* 创建自适应表单
* 复杂表单的调试
* 配置和执行表单提交
* 构建和部署用于自适应表单的AEM实例
* 使用表单模型
* 能够与数据服务集成
* 工作流实施
* 对AEM Forms Workbench(JEE)的基本了解，可创建实时性较长的流程和服务。
* 使用GuideBridge API开发javascript客户端库
* 基于sign v government CAC了解基于哪个AEM Forms环境桌面或服务器要使用
* 基本了解Java/J2EE和JavaScript以编写逻辑并调用不同的服务。
* 预填自适应表单
* 对OSGi、Sling组件的基本了解或了解
* 使用文档服务
* AEM Designer， AEM Workbench， AEM工作区
* 自适应表单控制台
* Java IDE工具
* Maven
* GIT/版本控制工具

>[!NOTE]
>
>您的证书有效期为2年，从您通过之日算起。 我们建议尽可能使用较新版本，因为它涵盖较新的产品功能。

**考试详细信息：**

* 级别：专家（1-3年经验）
* 通过分数：27/50
* 时间：100分钟
* 投放：在线监控（需要摄像头访问）或测试中心监控
* 可用语言：英语
* 成本：225美元（全球）/ 150美元（印度）
* 考试ID:AD0-E125

**准备情况自评**

完成 [在线准备调查表](https://scorpion.caveon.com/launchpad/ad-q-e129-readiness-questionnaire-for-adobe-aem-assets-developer-professional-exam-copy-x38d6m/ad-q-e125-readiness-questionnaire-for-adobe-aem-forms-developer-expert-exam){target="_blank"} 以查看您当前的体验是否与建议的最低候选资格匹配。

**考试目标和范围**

第1节：Forms创建和管理(36%)

* 构建Forms
* 配置Forms数据模型
* 自定义AEM Forms
* 利用GuideBridge API开发和使用Javascript ClientLib

第二节：流程和工作流(34%)

* 设计和编排表单批准工作流程
* 使用Document Services API开发OSGi包
* 使用Watchfolder端点计划不同的异步批处理
* 从XFA/XDP设计生成不同的输出
* 使用Forms或汇编程序服务处理PDF
* 实施JDBC服务

第三节：管理和故障排除(18%)

* 日志文件故障诊断和区分
* 执行管理活动
* 应用服务包
* 为Forms执行AEM Dispatcher设置
* 在不同的控制台中导航

第四节：API端点和集成(8%)

* 与 Adobe Experience Cloud 集成
* 生成和使用各种端点服务

第五节：交互式通信(4%)

* 设计、演示和实施交互式通信的流程
